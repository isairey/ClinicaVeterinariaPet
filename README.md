## 🐶 Descripción

Sistema de información para la **gestión de una clínica veterinaria**, desarrollado con tecnologías web y backend en Python.

Permite administrar mascotas, clientes, citas médicas, historial clínico y procesos administrativos de manera eficiente y organizada.

---

## 🚀 Características

- 🐾 Registro de mascotas  
- 👤 Gestión de propietarios  
- 📅 Administración de citas  
- 🩺 Historial clínico  
- 💊 Control de tratamientos  
- 📋 Registro de consultas  
- 📊 Reportes del sistema  
- 🔐 Autenticación de usuarios  

---

## 🧩 Módulos del Sistema

- 🐶 **Mascotas** → Registro y control de pacientes  
- 👤 **Clientes** → Información de propietarios  
- 📅 **Citas** → Agendamiento de consultas  
- 🩺 **Historial Clínico** → Diagnósticos y tratamientos  
- 💰 **Administración** → Control general del sistema  

---

## 🛠️ Tecnologías Utilizadas

- Python 🐍 (Backend)  
- HTML (Estructura web)  
- CSS (Estilos)  
- JavaScript (Interactividad)  
- Scripts Batch / Shell
